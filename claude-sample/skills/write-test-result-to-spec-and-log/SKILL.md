# テスト結果の明細書反映＆ログ記録スキル

[[execute-screen-test-and-log]] で仕様書記載のテスト項目をすべて実施し終えた後、
その判定結果（実施日・結果・備考等）をテスト明細書（xlsx）の「実施日」「結果」「備考・特記事項」等の
該当セルに書き戻し、書き込み内容を検証してログに記録するスキル。CLAUDE.mdの
「結果を報告する」「期待値を推測してはいけない」方針、および「やったことは全部ログに記録する
（実施前・実施中・実施後それぞれのタイミングで書き残す）」というユーザー方針に従う。

## 前提
- 仕様書記載のテスト項目すべてについて、[[execute-screen-test-and-log]] による実施・判定が完了していること
- 明細書（xlsx）への反映を行ってよいか、**事前にユーザーへ確認し承認を得ていること**
  （CLAUDE.mdの「ユーザーの指示なしに対象URLを変更すること」等と同様、原本ファイルへの書き込みは
  影響範囲が大きいため、必ず承認後に実施する）

## 実施手順

### 1. 反映前に「何をどう書き込むか」をログに記録する

ツールを実行する前に、当日のログファイル（例：`<ラウンド名>ログ/<YYYY-MM-DD>_..._仕様書結果反映ログ.md`）に
以下を記載する。**書き込みを実行する前に計画をログ化すること。**

- 経緯（どのテスト実施ログの結果を反映するか、ユーザーからどのような承認を得たか）
- 書き込み方法（`openpyxl.load_workbook(path)` を `data_only=True` を付けずに開き、
  書式・数式を保持したまま値だけを設定して `wb.save()` する旨）
- 書き込み対象のセル位置（行・列）と書き込む値の対応表
  - [[read-test-spec-and-log]] で確認した列構成（No／分類／テスト項目／検証内容／実施日／結果／備考・特記事項）
    と、各テスト項目が何行目に対応するかを明記する
  - 「結果」列には [[execute-screen-test-and-log]] で確定した判定（OK／NG等）をそのまま転記する
    （ここでも期待値や判定を新たに推測・変更しない）
  - 「備考・特記事項」列には、検証内容には含まれないが実施時に観測した事象
    （例：favicon.ico の404エラーなど、ユーザーに事実として報告済みの事象）がある項目にのみ記載する

### 2. openpyxlで明細書に書き込む

```python
import openpyxl
from datetime import datetime

wb = openpyxl.load_workbook(path)          # data_only=True は付けない（書式保持のため）
ws = wb["Sheet1"]

ws.cell(row=<行番号>, column=<実施日の列番号>, value=datetime(YYYY, M, D))
ws.cell(row=<行番号>, column=<結果の列番号>,  value="OK")
ws.cell(row=<行番号>, column=<備考の列番号>,  value="...")  # 特記事項がある場合のみ

wb.save(path)
```

- 列番号は [[read-test-spec-and-log]] で確認したヘッダー行の並び（0始まりindexを1始まりのExcel列番号に変換）に基づいて特定する
- 結合セル（`ws.merged_cells.ranges`）の範囲に重なっていないか事前に確認する

### 3. 書き込み結果を再読込して検証する

保存後、`load_workbook(path, data_only=True)` で読み直し、書き込んだ値が意図どおりに
反映されているかを確認する。確認用に一時ファイル（例：`verify_dump.txt`）にダンプしてもよいが、
**確認が終わったら必ず削除する**（[[read-test-spec-and-log]] の一時ダンプファイル削除ルールと同様）。

### 4. ログに結果を追記する

手順1で記録した「実施予定」に対応する形で、以下を追記する。

- 実際に書き込んだ内容（テスト項目ごとに：実施日／結果／備考・特記事項を表形式で整理）
- 再読込による検証結果（意図どおり反映されたか）
- 一時ファイルを削除した旨
- ステータス（明細書への反映が完了し検証済みである旨）

### 5. ユーザーへ報告する

反映した内容を表形式で簡潔に提示し、完了したことを報告する。
ログファイルへのリンクも添える。
